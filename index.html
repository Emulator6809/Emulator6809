About Emulator 6809

Born from a deep passion for computing science and education, Emulator 6809 is dedicated to one primary mission: promoting the study and understanding of microprocessor-based systems.

We believe that understanding the fundamentals of microprocessor architecture is essential for any serious computer scientist or engineer. The legendary Motorola 6809 processor, with its elegant design and powerful instruction set, serves as the perfect educational platform.

This project represents our commitment to making low-level computing accessible, engaging, and educational for students, hobbyists, and professionals around the globe. Through accurate emulation, intuitive tools, and comprehensive documentation, we're preserving computing heritage while inspiring the next generation of engineers.

Emulator

Main Window

The main interface provides a complete view of the MC6809 minimal computing system, displaying the CPU with all its registers, ROM, RAM, System Stack, and User Stack in real-time. It includes hardware interrupt simulation capabilities (IRQ, FIRQ, and NMI) for comprehensive system testing. The integrated code editor enables users to write assembly programs or open existing files, assemble them using the built-in assembler, and execute them with educational controls: RUN for continuous execution, STEP for single-instruction debugging, and HALT to stop execution. The editor highlights the currently executing line, providing clear visual feedback for learning program flow and debugging. This comprehensive interface is specifically designed for educational purposes, helping users understand how a minimal computing system operates with all its essential components: CPU 6809, ROM, RAM, and I/O devices.

System Configuration

Memory Configuration

The Memory Configuration interface provides intuitive memory management capabilities, allowing users to easily configure ROM size, RAM size, System Stack size, and User Stack size. This user-friendly interface simplifies memory layout customization, making it accessible for educational purposes while providing the flexibility needed for advanced development scenarios.

I/O Configuration

The I/O Configuration interface provides comprehensive peripheral address management, allowing users to configure memory-mapped addresses for all I/O devices. This flexibility enables custom system architectures and facilitates learning about memory-mapped I/O concepts in microprocessor-based systems.

I/O Peripherals

ASCII Screen Display

The ASCII Screen Display emulates a text-based output device with a 5-line by 20-character display matrix. Each character position can display any standard ASCII character, making it ideal for system messages, debugging output, and simple user interfaces in 6809-based applications.

PIA

The Peripheral Interface Adapter (PIA) offers two dedicated 8-bit ports for system I/O operations. One port is configured for input operations while the other handles output. Each bit within both ports can be individually managed and monitored in real-time, providing granular control over peripheral devices and enabling detailed observation of data flow in 6809-based applications.

BCD 7-Segment Digital Clock

The BCD 7-Segment Digital Clock provides a real-time clock interface that user programs can read and adjust. It supports two operating modes: Automatic mode, where the clock runs independently using system time, and Manual mode, where the user program must generate clock pulses to advance the time. This dual-mode operation enables both real-time applications and precise timing control for testing and educational demonstrations.

Analog Sensors Interface

The Analog Sensors Interface enables simulation of multiple analog sensors including voltage, temperature, luminosity, and pressure measurements. It features an integrated Analog-to-Digital Converter (ADC) with configurable resolution (8-bit or 16-bit) and selectable output format (BCD or natural binary). This flexibility allows users to experiment with various sensor configurations and data acquisition techniques in educational and development scenarios.
