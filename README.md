# Emulator 6809

**Emulator 6809** is a minimal computer system simulator based on the Motorola MC6809 microprocessor.  
It includes ROM, RAM, basic I/O, and an integrated **assembler**, designed for education, low-level software development, and experimentation with microprocessor-based systems.
